TECHNOLOGY

SPL Global Telemetry System

SPL is an open global satellite telemetry system for unmanned vehicles controlled by ArduPilot orPX4 autopilots that uses Iridium short burst data (ISBD) satellite communication technology provided by Rock Seven Mobile.

more...

NVI BVLOS Telemetry System

NVI is a beyond visual line of sight (BVLOS) telemetry system for unmanned vehicles that works with satellite or cellular modems such as Iridium GO! or Iridium Certus.
